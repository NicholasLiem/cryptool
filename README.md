## 🖥️ Running locally for development

1. Clone this repository
```sh
git@github.com:NicholasLiem/cryptool.git
```

2. Change the current directory to 'cryptool' folder
```sh
cd cryptool
```

3. Install gem dependencies
```sh
bundle install
```

4. Run migrations
```sh
rails db:migrate
```

5. Run the server
```sh
rails server
```
